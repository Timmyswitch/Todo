# To-Do List App

A clean, minimal task management app built with vanilla HTML, CSS, and JavaScript.

## Features

- Add, edit, and delete tasks
- Mark tasks as complete
- Filter tasks by All, Active, or Completed
- Mini dashboard showing total, active, and done counts
- Progress bar that tracks completion
- Tasks saved to localStorage — persists on refresh

## File Structure

```
├── index.html
├── style.css
├── index.js
└── assets/
    └── background-1.jpg
```

## Getting Started

No installs or dependencies needed. Just open `index.html` in a browser and you're good to go.
